Você pode cadastrar, alterar e excluir uma historia, quando começar a cadastrar, começe pelo id=0 e os campos id de destino da 1° escolha e id de destino da 2° escolha, tem que necessáriamente conter um id, como voçe começou a cadastrar é bem provável que não tenha os ids seguintes então preencha os dois com 0 que é o id iniciale depois pode alterar para id da historia que essa escolha leva, você pode alterar os ids das histórias desde que esse id já não esteja cadastrado, quando for excluir, pense bem, pois quando clicar na lixeira vai excluir toda a história.
